Introduction to Musubi

Musubi is the multi-level parallel lattice Boltzmann solver within the APES suite. It is working on a linearized octree and uses efficient data structures allowing adaptive parallel simulations. Musubi offers several collision kernels and is designed in a way to deal with huge meshes efficiently.

See the documentation for details, tutorials and examples.
