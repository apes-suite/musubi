This module contains function to compute eddy viscosity for Wall-Adapting Local Eddy-Viscosity turbulence model. This implementation follows the LES described by Weickert et al. Weickert, M., Teike, G., Schmidt, O., & Sommerfeld, M. (2010). Investigation of the LES WALE turbulence model within the lattice Boltzmann framework. Computers and Mathematics with Applications, 59(7), 2200â€“2214. author: Kannan Masilamani
