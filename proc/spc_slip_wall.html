slip-wall boundary condition. Slip defined by a slip factor

  • Normal velocity,
  • Tangential velocity,
  • Pressure,

For slip-wall boundary, the slip factor will be multiplied by the velocity if slip factor = 1, then it is full/free-slip and if slip factor = 0, then it is no-slip
