public subroutine applySrc_force_GNS(fun, inState, outState, neigh, auxField, nPdfSize, iLevel, varSys, time, phyConvFac, derVarPos)

Update state with source variable "force" for Generalized Navier-Stokes equations. The implementation is taken from "Lattice Boltzmann model for incompressible flows through porous media" by Z. Guo and T.S. Zhao (2002) Physical Review E.

This subroutine's interface must match the abstract interface definition proc_apply_source in derived/mus_source_var_module.f90 in order to be callable via applySrc function pointer.
