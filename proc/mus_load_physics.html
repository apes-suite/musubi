public subroutine mus_load_physics(me, conf, tree, scaleFactor, dtRef, dxRef)

This routine loads the physics table from musubi config file

If no physics table is provided, the conversion factors default to 1, resulting in the lattice units being directly used. dx_ref and dt_ref are set according to the provided arguments, or if not provided default to 1. See the [mus_physics_type] for a description of the various factors that can be set here.
