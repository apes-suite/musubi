public recursive subroutine deriveEquil_fromIndex(fun, varSys, time, iLevel, idx, idxLen, nVals, res)

Initiates the calculation of equilibrium. This routine sets the function Pointer for equilibrium calcualtion and calls the generice get Value of Index routine

The interface has to comply to the abstract interface tem_varSys_proc_getValOfIndex.
